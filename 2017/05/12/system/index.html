系统推荐

我又把它找回来了

闲的没事干（怎么不去学技术），想写技术贴，但是目前没有什么技术，所以推荐几个好玩的系统吧
Windows的系统就不去说了，大家日常都会用，介绍了也没什么意思，主要介绍几种Unix或者类Unix系统。

Linux Mint

Linux Mint是一份基于Debian和Ubuntu的Linux发行版，它并不像许多为我们所熟知的Liunx系统，内置许多安全工具，而是一种面向家庭用户和企业的简单易用，舒适优雅的桌面操作系统。
你可以在虚拟机中安装，也可以直接将他安装在物理机中，它会自动检测其他系统并建立双系统启动。
如果有想体验一下Linux系统的小伙伴可以尝试一下。

Kali Linux

Kali是基于Debian的一个Linux的发行版，它的前身是Backtrack，系统包含很多安全和取证方面的相关工具，是一款很适合做安全和攻击的系统。相信Kali是很多黑客的最爱。

Weakerthan Linux

Weakerthan Linux是一款基于Debian的压缩版渗透测试系统，它包含有大量的无线工具，很适合WiFi攻击，蓝牙攻击等，重点是它的桌面看起来很炫酷。
PS.如果有小伙伴想用，建议用Weakerthan Linux 6版本，在官网上有详细的虚拟机安装视频，至于7版本目前还只是一个beta测试版，并不完善。（有墙自翻）

Parrot Security

Parrot Security是一款基于Debian的发行版，它包括加密、云、匿名、数字取证分析和编程等方面的软件，同样，颜值很高。

Others

好的系统还有许多，比如LionSec Linux，BackBox等，我也不是都用过，不好妄加评论，大家有兴趣可以自己去体验一下，说不定会有意想不到的发现。

最后想说，我对每个系统的理解可能与大家不同，有什么不对或者需要指出的地方，欢迎大家联系我，邮箱放在这里：reiboy@yahoo.com

在我看来，喜欢上一个系统，与喜欢上一个人是一样的，最初也许因为好看、易用一些原因去接触，但是真正去喜欢上它还是需要时间的，喜欢上了，也就放不下了。就像我初用Kali时，是为了里面的安全工具，觉得它难用，比不上图形化程度很高的Windows，但是用的时间越久，我就越是喜欢用Kali，尤其是我的Kali，甚至是让我卸掉重装我也是舍不得的。
